Followed words

A followed word is a shorter word entry that can be transformed into another, longer entry by adding letters to its end.

Word entries that are not followed score one point, and add time bonuses equal to their length minus three. The length of each time bonus is determined by the Pace slider on the Setup dialog.

In this example, the word fine has been entered:

Entry Points Time bonuses
fine 1 1

If finest is entered next, fine becomes a followed word, since it can be changed to finest by adding st. The first entry is invalidated, and one point is awarded for the second, along with three time bonuses:

Entry Points Time bonuses
fine 0 0
finest 1 3

If fines is entered, it is immediately invalidated, and no point is scored. Though it follows fine, it is itself already followed by finest:

Entry Points Time bonuses
fine 0 0
finest 1 3
fines 0 0

For an entry to be followed, it must be possible to transform it to another entry by adding letters; simply sharing a root is not enough. Thus, entering finer scores a point without invalidating anything, though it would have invalidated fine if that word had not already been followed:

Entry Points Time bonuses
fine 0 0
finest 1 3
fines 0 0
finer 1 2

A word is followed only when letters are added to its end, so entering refiner scores a point without invalidating finer:

Entry Points Time bonuses
fine 0 0
finest 1 3
fines 0 0
finer 1 2
refiner 1 4

Finally, only player-entered words cause other words to be followed. If finery is spelled within the grid, and if the player fails to find it, finer will remain valid, even though Ogle finds and scores finery itself.

These rules becomes intuitive with practice. Followed words are ignored so you don't have to enter the same string repeatedly when longer words (especially plurals) are encountered. When you find a group of related words, just enter the longest one, and you'll get as many points and as much bonus time as is possible.
